# TeamGossip-Git

Lancement du projet :

Ouvrir un terminal dans le dossier du projet.

Exécuter la commande : go run main.go

Se rendre sur le lien local affiché dans la console.

Fonctionnement :

Une fois sur l’interface, choisir une thématique de quiz.

Chaque question propose une seule réponse possible, sans choix multiples.

À la fin du quiz, un score sur 10 est affiché.

Évaluation :

Un score minimum de 7/10 est requis.

Si le score est inférieur à 7, le message "Maîtrise insuffisante" s’affiche.
